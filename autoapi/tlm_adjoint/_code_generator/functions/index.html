tlm_adjoint._code_generator.functions

This module is used by both the FEniCS and Firedrake backends, and includes functionality for handling UFL Coefficient objects and boundary conditions.

Module Contents

class tlm_adjoint._code_generator.functions.Constant(value=None, *args, name=None, domain=None, space=None, space_type='primal', shape=None, comm=None, static=False, cache=None, checkpoint=None, **kwargs)

Extends the backend Constant class.

Parameters

  • value – The initial value. None indicates a value of zero.

  • name – A str name.

  • domain – The domain on which the Constant is defined.

  • space – The space on which the Constant is defined.

  • space_type – The space type for the Constant. ‘primal’, ‘dual’, ‘conjugate’, or ‘conjugate_dual’.

  • shape – A tuple of int objects defining the shape of the value.

  • comm – An mpi4py.MPI.Comm, communicator for the Constant.

  • static – Defines the default value for cache and checkpoint.

  • cache – Defines whether results involving this Constant may be cached. Default static.

  • checkpoint – Defines whether a tlm_adjoint.checkpointing.CheckpointStorage should store this Constant by value (checkpoint=True) or reference (checkpoint=False). Default not static.

Remaining arguments are passed to the backend Constant constructor.

tlm_adjoint._code_generator.functions.eliminate_zeros(expr, *, force_non_empty_form=False)

Apply zero elimination for Zero objects in the supplied UFL Expr or Form.

Parameters

  • expr – A UFL Expr or Form.

  • force_non_empty_form – If True and if expr is a UFL Form, then the returned form is guaranteed to be non-empty, and may be assembled.

Returns
