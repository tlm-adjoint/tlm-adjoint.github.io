tlm_adjoint.checkpoint_schedules.checkpoint_schedulesÔÉÅ

Translation between checkpointing schedules provided by the checkpoint_schedules library and a tlm_adjoint CheckpointSchedule.

Wrapped checkpoint_schedules.schedule.CheckpointSchedule classes can be imported from this module and then passed to configure_checkpointing(), e.g.

from tlm_adjoint import configure_checkpointing
from tlm_adjoint.checkpoint_schedules.checkpoint_schedules \
    import MultistageCheckpointSchedule

configure_checkpointing(
    MultistageCheckpointSchedule,
    {"max_n": 30, "snapshots_in_ram": 0, "snapshots_on_disk": 3})
