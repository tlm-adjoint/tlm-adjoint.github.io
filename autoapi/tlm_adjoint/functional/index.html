class tlm_adjoint.functional.Functional(*, space=None, name=None, _fn=None)

A convenience class for defining functionals.

This allocates and stores an internal function, but note that this function can change e.g. after adding terms.

Parameters

  • space – The space for the Functional. Internal functions are in this space. Defaults to FloatSpace(Float).

  • name – A str name for the functional.

assign(self, term, *, annotate=None, tlm=None)

Assign to the functional,

\[\mathcal{J} = b.\]

Note that this method allocates a new internal function.

Parameters

  • term – The value. Defines the value of \(b\). Valid types depend upon the backend. \(b\) may be a function, and with the FEniCS or Firedrake backends may be an arity zero UFL Form.

  • annotate – Whether the tlm_adjoint.tlm_adjoint.EquationManager should record the solution of equations.

  • tlm – Whether tangent-linear equations should be solved.

addto(self, term=None, *, annotate=None, tlm=None)

Add to the functional. Performs two assignments,

\[\mathcal{J}_\text{term} = b,\]
\[\mathcal{J}_{new} = \mathcal{J}_\text{old} + \mathcal{J}_\text{term},\]

where \(\mathcal{J}_\text{old}\) and \(\mathcal{J}_\text{new}\) are, respectively, the old and new values for the functional, and \(b\) is the term to add.

Note that this method allocates a new internal function.

Parameters

  • term – The value. Defines the value of \(b\). Valid types depend upon the backend. \(b\) may be a function, and with the FEniCS or Firedrake backends may be an arity zero UFL Form.

  • annotate – Whether the tlm_adjoint.tlm_adjoint.EquationManager should record the solution of equations.

  • tlm – Whether tangent-linear equations should be solved.

tlm_functional(self, *args)

Return a Functional associated with a tangent-linear variable associated with the functional.

Parameters

args – A Sequence of (M, dM) pairs. Here M and dM are each a function or a sequence of functions defining a derivative and derivative direction. The tangent-linear variable is the derivative of the functional with respect to each M and with direction defined by each dM. Supplying a single pair requests a Functional associated with a first order tangent-linear variable. Supplying multiple pairs leads to a Functional associated with higher order tangent-linear variables. The relevant tangent-linear models must have been configured for the tlm_adjoint.tlm_adjoint.EquationManager manager.

Returns

A Functional associated with the tangent-linear variable.
