abstract action(self, M, dM, M0=None)

Compute (the conjugate of) a Hessian matrix action on some \(\zeta\) using an adjoint of a tangent-linear model. i.e. considering derivatives to be column vectors, compute

\[\left( \frac{d}{dm} \left[ \frac{d \mathcal{J}}{d m}^T \zeta \right] \right)^{*,T}.\]
Parameters

  • M – A function or a Sequence of functions defining the control variable.

  • dM – A function or a Sequence of functions defining \(\zeta\). The (conjugate of the) Hessian matrix action on \(\zeta\) is computed.

  • M0 – A function or a Sequence of functions defining the control value. M is used if not supplied.

Returns

A tuple (J, dJ, ddJ). J is the value of the functional. dJ is the value of \(\left( d \mathcal{J} / d m \right)^T \zeta\). ddJ stores the (conjugate of the) result of the Hessian matrix action on \(\zeta\), and is a function or a Sequence of functions depending on the type of M.

action_fn(self, m, m0=None)

Return a callable which can be used to compute Hessian matrix actions.

Parameters

  • m – A function defining the control variable.

  • m0 – A function defining the control value. m is used if not supplied.

Returns

A callable which accepts a single function argument, and returns the result of the Hessian matrix action on that argument as a function. Note that the result is not the conjugate of the Hessian matrix action on the input argument.

class tlm_adjoint.hessian.GaussNewton(R_inv_action, B_inv_action=None, *, J_space=None)

Represents a Gauss-Newton approximation for a Hessian matrix. Abstract base class.

In terms of matrices this defines a Hessian matrix approximation

\[H = J^* R_\text{obs}^{-1} J + B^{-1},\]

where \(J\) is the forward Jacobian. In a variational assimilation approach \(R_\text{obs}^{-1}\) corresponds to the observational inverse covariance and \(B^{-1}\) corresponds to the background inverse covariance.

Parameters

  • R_inv_action – A callable which accepts one or more functions, and returns the conjugate of the action of the operator corresponding to \(R_\text{obs}^{-1}\) on those functions, returning the result as a function or a Sequence of functions.

  • B_inv_action – A callable which accepts one or more functions, and returns the conjugate of the action of the operator corresponding to \(B^{-1}\) on those functions, returning the result as a function or a Sequence of functions.

  • J_space – The space for the functional. FloatSpace(Float) is used if not supplied.

action(self, M, dM, M0=None)

Compute (the conjugate of) a Hessian matrix action on some \(\zeta\), using the Gauss-Newton approximation for the Hessian matrix. i.e. compute

\[\left( H \zeta \right)^{*,T}.\]
Parameters

  • M – A function or a Sequence of functions defining the control variable.

  • dM – A function or a Sequence of functions defining \(\zeta\). The (conjugate of the) approximated Hessian matrix action on \(\zeta\) is computed.

  • M0 – A function or a Sequence of functions defining the control value. M is used if not supplied.

Returns

The (conjugate of the) result of the approximated Hessian matrix action on \(\zeta\). A function or a Sequence of functions depending on the type of M.
