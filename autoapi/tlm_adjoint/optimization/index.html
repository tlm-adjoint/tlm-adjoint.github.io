Defines the initial Hessian inverse approximation. Accepts one or more variables as arguments, defining a direction, and returns a variable or a Sequence of variables defining the (conjugate of) the action of an initial Hessian inverse approximation on this direction. Arguments should not be modified.

Defines a dual space norm. Accepts one or more variables as arguments, defining a direction, and returns a variable or a Sequence of variables defining the (conjugate of) the action of a Hermitian and positive definite matrix on this direction. Arguments should not be modified. H_0_action is used if not supplied.
