class tlm_adjoint.tangent_linear.TangentLinearMap(M, dM)

Defines a map from forward variables to associated tangent-linear variables.

The map is used via e.g.

tau_x = tlm_map[x]

where x is a function associated with the forward variable. If the function x is ‘checkpointed’, meaning that it is stored by value by a tlm_adjoint.checkpointing.CheckpointStorage, then tau_x is a function associated with the tangent-linear variable – a new function is instantiated if needed. If the function x is not ‘checkpointed’, meaning that it is stored by reference by a tlm_adjoint.checkpointing.CheckpointStorage, then tau_x is None.

Containment can also be tested

if x in tlm_map:
    [...]

and returns True if a tangent-linear function associated with x has been instantiated.

Note that a TangentLinearMap should not be used to map from the control M to the derivative direction dM. Typically a TangentLinearMap should not be used directly, and instead get_tangent_linear() should be used, which does map from the control to the direction.

Parameters

  • M – A function or Sequence of functions defining the control.

  • dM – A function or Sequence of functions defining the derivative direction. The tangent-linear model computes directional derivatives with respect to the control defined by M and with direction defined by dM.

tlm_adjoint.tangent_linear.get_tangent_linear(x, M, dM, tlm_map)

Return a tangent-linear variable associated with a variable x.

This function should be used in place of accessing via the TangentLinearMap, if the variable x may be a control variable.

Parameters

  • x – A function defining the variable for which a tangent-linear variable should be returned.

  • M – A Sequence of functions defining the control.

  • dM – A Sequence of functions defining the derivative direction. The tangent-linear model computes directional derivatives with respect to the control defined by M and with direction defined by dM.

  • tlm_map – A TangentLinearMap storing values for tangent-linear variables.

Returns

If x is a control variable then returns the associated direction. If x is not a control variable then returns a function corresponding to a tangent-linear variable if x is ‘checkpointed’ (i.e. stored by value by a tlm_adjoint.checkpointing.CheckpointStorage), and None otherwise.
